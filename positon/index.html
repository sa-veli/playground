  • Absolute

Header one

I start with basics. position: absolute; is a simple thing. Use top: [value]; or other side's name to change where your element position itself. If value is zero than it will placed in top left corner of the page. Use position: relative; on ancestor element to positon relative to it. See example below.

Header two

position: relative; Element positioned according to normal flow of the document and then offset relative to itself based on values of top bottom and so on.

Divs are block-level elements that's why boxes below are not placed on the same line.

Header three

positon: fixed; it offsets relative to viewport and stays on screen even when we scroll. Box stuck to the right side of the page has this declaration.

This box has position: fixed. See third section to understand what is going on with it.

Header four

position: sticky; This rule declaration made element stick to on of the sides of viewport. And we can make it offset. If I write bottom: 0; it will stay on bottom. Headers and navigation menu on this page use this declaration. Notice how headers replace each other but navigation menu stays on top. This happens because nav menu is part of <main> and headers are part of their respective sections.
